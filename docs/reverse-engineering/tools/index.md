# Reverse Engineering Tools

## Surface Reconstruction

| Tool | Description |
|------|-------------|
| [Poisson…](surface-reconstruction.md#poisson-reconstruction) | Reconstruct a watertight mesh from a point cloud using the Poisson algorithm |
| [Structured Point Clouds](surface-reconstruction.md#structured-point-clouds) | Triangulate a structured (grid) point cloud into a mesh |

## Segmentation

| Tool | Description |
|------|-------------|
| [Mesh Segmentation…](segmentation.md#mesh-segmentation) | Automatically segment a mesh into regions by surface type |
| [Manual Segmentation…](segmentation.md#manual-segmentation) | Paint mesh regions interactively to define segments |
| [From Components](segmentation.md#from-components) | Create one mesh segment per disconnected component |
| [Wire From Mesh Boundary…](segmentation.md#wire-from-mesh-boundary) | Extract boundary loops of a mesh as Part wires or faces |

## Approximation

| Tool | Description |
|------|-------------|
| [Plane](approximation.md#plane) | Fit a best-fit plane to geometry |
| [Cylinder](approximation.md#cylinder) | Fit a best-fit cylinder to a mesh |
| [Sphere](approximation.md#sphere) | Fit a best-fit sphere to a mesh |
| [Polynomial Surface](approximation.md#polynomial-surface) | Fit a degree-2 Bézier surface patch to a mesh |
| [Approximate B-Spline Surface…](approximation.md#approximate-b-spline-surface) | Fit a free-form B-spline surface to a point cloud or mesh |
| [Approximate B-Spline Curve…](approximation.md#approximate-b-spline-curve) | Fit a B-spline curve to a point cloud |
