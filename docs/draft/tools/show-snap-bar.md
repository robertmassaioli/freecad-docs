# Show Snap Bar

> **In one sentence:** Re-displays the Draft Snap toolbar if it has been
> accidentally closed.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Utilities → Show Snap Bar  
**Command:** `Draft_ShowSnapBar`

The snap toolbar can be accidentally closed during a drawing session. This
command restores it without restarting FreeCAD.
