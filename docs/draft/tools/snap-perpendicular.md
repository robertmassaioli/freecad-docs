# Snap Perpendicular

> **In one sentence:** Snaps to the point on an edge that is perpendicular
> to the previous drawn point, ensuring the new segment is at 90° to the edge.

---

## Overview

**Workbench:** Draft  
**Toolbar:** Draft Snap  
**Command:** `Draft_Snap_Perpendicular`

**Indicator:** Yellow cross with perpendicularity symbol.

Useful for drawing lines that must be exactly normal to an existing edge.
