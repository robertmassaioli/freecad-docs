# Facebinder

> **In one sentence:** Creates a parametric compound of selected faces from
> existing 3-D solids, updating when the source solids change shape.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Drafting → Facebinder  
**Command:** `Draft_Facebinder`

Select one or more faces (hold **Ctrl** to multi-select), then activate the
tool. The facebinder creates a flat reference surface from those faces that
follows the solid as it changes shape — parametric.

---

## Use cases

- Creating flat reference surfaces from solid faces for Draft annotation.
- Applying a different material to a specific face.
- Extracting face geometry for further 2-D processing.
