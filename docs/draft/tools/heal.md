# Heal

> **In one sentence:** Attempts to repair malformed Draft objects — typically
> those created in older FreeCAD versions with deprecated internal properties.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Utilities → Heal  
**Command:** `Draft_Heal`

If geometry appears missing or a Draft object throws errors in the model tree,
try running Heal on it. The tool upgrades objects to the current Draft
object format.
