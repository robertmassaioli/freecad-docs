# Select Group

> **In one sentence:** Selects all objects in the same group or layer as the
> currently selected object.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Utilities → Select Group  
**Command:** `Draft_SelectGroup`

Useful for quickly selecting all geometry on a layer before moving it,
changing its style, or deleting it.
