# Point Link Array

> **In one sentence:** Like Point Array but uses App::Link instances —
> lighter on memory for large point arrays.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Modification → Array Tools → Point Link Array  
**Command:** `Draft_PointLinkArray`

Identical to [Point Array](array-point.md) but each element is an `App::Link`.
Preferred for large point arrays (50+ elements).
