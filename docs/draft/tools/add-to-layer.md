# Add to Layer

> **In one sentence:** Moves selected objects to a chosen layer, applying
> that layer's visual properties.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Utilities → Add to Layer  
**Command:** `Draft_AddToLayer`

A submenu lists all existing layers. Select one or more objects, then pick
the target layer.
