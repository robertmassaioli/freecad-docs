# Stretch

> **In one sentence:** Moves selected vertices of Draft wires while
> maintaining connectivity with unselected vertices.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Modification → Stretch  
**Command:** `Draft_Stretch`

Useful for stretching one end or one section of a drawing while keeping
the rest fixed in place.

---

## Step-by-step

1. Activate **Draft → Modification → Stretch**.
2. Draw a **selection rectangle** around the vertices to move.
3. Pick the **base point** for the stretch displacement.
4. Pick the **target point**.

Only vertices inside the selection rectangle are moved. Adjacent segments
to unselected vertices are stretched to maintain connectivity.
