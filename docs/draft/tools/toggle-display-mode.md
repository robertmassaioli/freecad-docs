# Toggle Display Mode

> **In one sentence:** Switches selected Draft objects between Flat Lines
> (filled faces with edge lines) and Wireframe (edges only) display modes.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Utilities → Toggle Display Mode  
**Command:** `Draft_ToggleDisplayMode`

| Mode | Description |
|------|-------------|
| Flat Lines | Show filled faces with edge lines (default) |
| Wireframe | Show edges only — no face fill |

Useful for quickly making overlapping faces transparent to see geometry
underneath without permanently changing the object's properties.
