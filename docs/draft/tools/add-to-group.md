# Add to Group

> **In one sentence:** Adds selected objects to an existing named group
> for logical organisation in the model tree.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Utilities → Add to Group  
**Command:** `Draft_AddToGroup`

A submenu lists all existing groups. No visual style change — groups are for
logical organisation only. For visual style grouping, use [Layer](layer.md).
