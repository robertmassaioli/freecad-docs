# Snap Extension

> **In one sentence:** Snaps along the extension of an existing edge beyond
> its endpoint in the same direction.

---

## Overview

**Workbench:** Draft  
**Toolbar:** Draft Snap  
**Command:** `Draft_Snap_Extension`

**Indicator:** Dashed yellow line showing the extension direction.

Move the cursor near an edge to lock in the extension direction, then move
along it to place the snap point beyond the endpoint.

---

## Use cases

- Placing objects in line with existing geometry but offset beyond the
  existing endpoints.
- Constructing extended reference lines without drawing them.
- Combined with [Snap Parallel](snap-parallel.md): snap along an extension at
  a position parallel to another edge (very powerful combination).
