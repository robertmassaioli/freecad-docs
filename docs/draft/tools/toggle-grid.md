# Toggle Grid

> **In one sentence:** Shows or hides the Draft grid in the 3-D view — a
> visual reference plane at the current working plane level.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Utilities → Toggle Grid  
**Command:** `Draft_ToggleGrid`

Also available in the **Snap toolbar** for quick access while drawing.
The grid is a visual reference only — it does not constrain snapping.
Use [Snap Grid](snap-grid.md) to snap to grid intersections.

---

## Grid settings

Configure in **Edit → Preferences → Draft → Grid**:

| Setting | Description |
|---------|-------------|
| Grid spacing | Distance between grid lines (mm) |
| Main lines every N | Bold grid lines at every Nth interval |
| Grid size | Number of grid squares visible |
| Grid color | Colour of grid lines |
