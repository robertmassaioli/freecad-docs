# Slope

> **In one sentence:** Sets the slope (rise/run ratio) of segments in a Draft
> Wire or Line by modifying each vertex's Z coordinate along the wire's length.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Modification → Slope  
**Command:** `Draft_Slope`

Used for drawing graded paths, drainage slopes, or ramps where the horizontal
projection is drawn flat but must have a specific gradient. The tool modifies
Z values along the wire proportionally.
