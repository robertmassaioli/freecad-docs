# Snap Special

> **In one sentence:** Snaps to object-specific special points not covered by
> other snap types — such as BIM insertion points, ShapeString character anchors,
> and structural member origins.

---

## Overview

**Workbench:** Draft  
**Toolbar:** Draft Snap  
**Command:** `Draft_Snap_Special`

| Object | Special points |
|--------|---------------|
| Part Design Body | Origin, placement point |
| Arch / BIM Wall | Corner points, opening positions |
| BIM elements | Structural insertion points |
| Draft ShapeString | Character insertion points |
