# Snap Grid

> **In one sentence:** Snaps the cursor to intersections of the Draft grid
> lines — ideal for floor plans and schematics that align to a regular grid.

---

## Overview

**Workbench:** Draft  
**Toolbar:** Draft Snap  
**Command:** `Draft_Snap_Grid`

**Indicator:** Small yellow cross at the nearest grid point.

Grid spacing is set in **Edit → Preferences → Draft → Grid → Grid spacing**.
