# Working Plane Proxy

> **In one sentence:** Saves the current working plane as a proxy object in
> the model tree — double-clicking it restores that working plane.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Utilities → Working Plane Proxy  
**Command:** `Draft_WorkingPlaneProxy`

Useful for documents with multiple views from different working planes.
Save each plane as a proxy and switch between them by double-clicking.
