# Wire to B-Spline

> **In one sentence:** Converts a Draft Wire (polyline) into a smooth Draft
> B-Spline through the same vertex positions, or vice versa.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Modification → Wire to B-Spline  
**Command:** `Draft_WireToBSpline`

Select a Wire to convert it to a B-Spline. Select a B-Spline to convert it
back to a polyline using the control points as vertices.
