# Add Named Group

> **In one sentence:** Prompts for a name and creates a new group container
> in the model tree for logical organisation of objects.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Utilities → Add Named Group  
**Command:** `Draft_AddNamedGroup`

Groups provide logical organisation in the model tree with no visual effect.
For visual grouping with style inheritance, use [Layer](layer.md) instead.

---

## Layers vs Groups

| Concept | Purpose | Visual effect |
|---------|---------|---------------|
| Layer | Visual styling for all members | Yes — colour, line width, draw style |
| Group | Logical organisation | None |
