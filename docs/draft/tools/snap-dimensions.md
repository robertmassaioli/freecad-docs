# Snap Dimensions

> **In one sentence:** Shows live Δx, Δy, Δz dimension labels next to the
> cursor while drawing, so you can verify distances without checking the
> coordinate input boxes.

---

## Overview

**Workbench:** Draft  
**Toolbar:** Draft Snap  
**Command:** `Draft_Snap_Dimensions`

Not a snap mode — rather, a display toggle that shows the distance from the
previous drawn point to the current cursor position in real time.

**Indicator:** Small dimension labels near the cursor showing Δx, Δy, Δz.
