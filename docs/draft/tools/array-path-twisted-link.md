# Path Twisted Link Array

> **In one sentence:** Like Path Twisted Array but uses App::Link instances —
> lighter on memory for large twisted path arrays.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Modification → Array Tools → Path Twisted Link Array  
**Command:** `Draft_PathTwistedLinkArray`

Identical to [Path Twisted Array](array-path-twisted.md) but each element
is an `App::Link`. Preferred for large arrays.
