# Apply Style

> **In one sentence:** Applies the current tool style (from Set Style) to the
> selected existing objects.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Utilities → Apply Style  
**Command:** `Draft_ApplyStyle`

Use after changing the tool style with [Set Style](set-style.md) to update
existing objects to match the new style.
