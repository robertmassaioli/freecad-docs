# Subelement Highlight

> **In one sentence:** Temporarily highlights individual sub-elements (edges,
> vertices) of objects to make them easier to select or snap to.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Modification → Subelement Highlight  
**Command:** `Draft_SubelementHighlight`

Useful when working with dense assemblies where selecting the correct
edge or vertex is difficult.
