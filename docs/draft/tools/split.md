# Split

> **In one sentence:** Splits a Draft wire at a selected existing vertex
> into two separate wires.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Modification → Split  
**Command:** `Draft_Split`

The split point must be an existing vertex of the wire (not a midpoint).
Select the wire, activate the tool, then click the vertex where you want
to split.
