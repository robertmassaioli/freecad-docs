# Offset

> **In one sentence:** Creates a wire parallel to the selected wire at a
> uniform offset distance, inward or outward.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Modification → Offset  
**Command:** `Draft_Offset`

The direction (inward or outward) is determined by the side you click.
The offset distance is uniform across all segments.

For complex shapes with variable offset, use **Part Workbench → Offset 2D**.

---

## Behaviour

- **Open wires:** Creates a parallel wire shifted by the offset distance.
- **Closed wires / faces:** Creates a larger or smaller version.
