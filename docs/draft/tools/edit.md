# Edit

> **In one sentence:** Opens a live editing session for a Draft object,
> showing its control points as draggable markers in the 3-D view.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Modification → Edit  
**Command:** `Draft_Edit`

---

## Supported objects

Draft_Edit works with: Wire, BSpline, BezCurve, Circle, Arc, Ellipse,
Rectangle, Polygon, Text, Dimension, Label, and Part objects (vertices only).

---

## Step-by-step

1. Select the Draft object.
2. Activate **Draft → Modification → Edit**.
3. Control points appear as small markers.
4. Drag a marker to reposition that point.
5. Right-click on a segment or point for context options:
   Add point, Delete point, Open/Close wire, etc.
6. Press **Escape** or click outside to exit edit mode.
