# Point Array

> **In one sentence:** Places one copy of the source object at each point
> vertex defined by a Draft Wire or Points object.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Modification → Array Tools → Point Array  
**Command:** `Draft_PointArray`

---

## Workflow

1. Create a Draft Wire marking the desired positions (each vertex = one
   array element position).
2. Select the object to array.
3. Activate **Draft → Modification → Array Tools → Point Array**.
4. Select the point wire (or compound of points).
