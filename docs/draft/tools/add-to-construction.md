# Add to Construction

> **In one sentence:** Moves selected objects to the Construction group and
> applies the construction colour.

---

## Overview

**Workbench:** Draft  
**Menu:** Draft → Utilities → Add to Construction  
**Command:** `Draft_AddConstruction`

The inverse of removing an object from construction mode. Use when you have
existing normal objects that you want to reclassify as construction references.
