# Snap Endpoint

> **In one sentence:** Snaps the cursor to the endpoints of edges and wires —
> the most-used snap type for precise geometry construction.

---

## Overview

**Workbench:** Draft  
**Toolbar:** Draft Snap  
**Command:** `Draft_Snap_Endpoint`

**Indicator:** Yellow square at the endpoint.

Snaps to the start and end vertices of line segments, arc endpoints, and
spline ends. Enable for all general drafting.
