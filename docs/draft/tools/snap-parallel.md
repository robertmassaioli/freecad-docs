# Snap Parallel

> **In one sentence:** Snaps along a direction parallel to an existing edge —
> hover the edge to lock its direction, then snap anywhere on parallel lines.

---

## Overview

**Workbench:** Draft  
**Toolbar:** Draft Snap  
**Command:** `Draft_Snap_Parallel`

**Indicator:** Dashed yellow line showing the parallel direction.

Move the cursor near an edge to lock in its direction, then move anywhere
to snap on lines parallel to that edge.

---

## Power combination

[Snap Extension](snap-extension.md) + Snap Parallel: snap along the extension
of one edge at a position parallel to another — useful for constructing
reference geometry without extra lines.
