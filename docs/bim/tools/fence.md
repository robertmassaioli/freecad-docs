# Fence

> **In one sentence:** Create a fence element by repeating a post shape and
> connecting with rail elements along a path wire.

---

## Overview

**Workbench:** BIM  
**Menu:** 3D/BIM → Fence  
**Command:** `Arch_Fence`  
**IFC class:** `IfcFence`

Creates a fence by repeating a post shape at regular intervals along a path
wire, connecting them with rail shapes. Both the post and rail are Part shapes
that you supply.

---

## Key properties

| Property | Description |
|----------|-------------|
| `Post` | Part shape used as the fence post |
| `Guard` | Part shape used as the fence rail/guard |
| `Spacing` | Distance between posts (mm) |
