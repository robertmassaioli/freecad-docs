# Dimension (Vertical)

> **In one sentence:** Place a dimension annotation that measures only the
> vertical (Z-axis) component of the distance between two points.

---

## Overview

**Workbench:** BIM  
**Menu:** Annotation → Dimension (Vertical)  
**Command:** `BIM_DimensionVertical`

Creates a vertical dimension — measures only the vertical (Z) component of
the distance between two points. Displays the height measurement with a
dimension line parallel to the Z-axis.

---

## When to use it

- Section and elevation views where you need storey heights.
- Annotating element heights (wall height, door height, etc.).
