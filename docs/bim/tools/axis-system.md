# Axis System

> **In one sentence:** Combine two or three Axis objects into a 2-D or 3-D
> column grid with labelled intersection points.

---

## Overview

**Workbench:** BIM  
**Menu:** Annotation → Axis System  
**Command:** `Arch_AxisSystem`

Combines two or three [Axis](axis.md) objects into a 2-D or 3-D column grid.
The grid points (intersections of axes) can be used to place columns and other
repetitive elements at precise grid positions.

---

## Step-by-step

1. Create two or three [Axis](axis.md) objects in different directions.
2. Select all of them.
3. Run **Annotation → Axis System**.
4. The grid intersections are computed and displayed.
