# Column

> **In one sentence:** Create a vertical structural column at a specified
> base point using a profile cross-section.

---

## Overview

**Workbench:** BIM  
**Menu:** 3D/BIM → Column  
**Command:** `BIM_Column`  
**IFC class:** `IfcColumn`

Creates a vertical structural column at a clicked base point. The cross-section
is defined by selecting a **Profile** from the profile library or a custom
sketch. Height and placement are set interactively.

---

## Step-by-step

1. Go to **3D/BIM → Column**.
2. Click in the 3-D view to place the column base point.
3. In the task panel, set the profile, height, and rotation.
4. Click OK.

---

## When to use it

- Structural steel or concrete columns in a framed building.
- Column grids referenced from an [Axis System](axis-system.md).
