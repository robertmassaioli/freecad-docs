# Beam

> **In one sentence:** Create a horizontal or inclined structural beam between
> two points using a profile cross-section.

---

## Overview

**Workbench:** BIM  
**Menu:** 3D/BIM → Beam  
**Command:** `BIM_Beam`  
**IFC class:** `IfcBeam`

Creates a structural beam between two clicked points. The profile cross-section
is chosen from the profile library. The beam follows the line between the two
points, sweeping the profile along the path.

---

## When to use it

- Floor beams, roof rafters, and structural frame members.
- Any linear structural element that is not a column or slab.
