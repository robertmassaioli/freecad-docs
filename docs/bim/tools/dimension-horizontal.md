# Dimension (Horizontal)

> **In one sentence:** Place a dimension annotation that measures only the
> horizontal (X-axis) component of the distance between two points.

---

## Overview

**Workbench:** BIM  
**Menu:** Annotation → Dimension (Horizontal)  
**Command:** `BIM_DimensionHorizontal`

Creates a horizontal dimension — measures only the horizontal component of
the distance between two points, regardless of any vertical offset between
them. Displays the horizontal measurement with a dimension line parallel to
the X-axis.

---

## When to use it

- Floor plan dimensions where you need the horizontal run (not the slant distance).
- Annotating horizontal spacing between elements.
