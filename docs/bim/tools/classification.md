# Classification

> **In one sentence:** Assign a classification code (Uniclass, OmniClass,
> ETIM, etc.) to elements for cost estimating and facilities management.

---

## Overview

**Workbench:** BIM  
**Menu:** Manage → Classification  
**Command:** `BIM_Classification`

Assigns a classification code to selected elements. Classification codes
(Uniclass 2015, OmniClass, ETIM, NBS, etc.) are stored as
`IfcClassificationReference` objects in the IFC file.

---

## Why classifications matter

Classification codes are used by:

- **Cost estimating** software to look up unit rates.
- **Facilities management** software to identify maintainable elements.
- **BIM compliance** checking (some contracts require Uniclass coding).
