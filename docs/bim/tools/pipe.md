# Pipe

> **In one sentence:** Create a circular MEP pipe along a wire or edge for
> plumbing, HVAC, or drainage modelling.

---

## Overview

**Workbench:** BIM  
**Menu:** 3D/BIM → Pipe  
**Command:** `Arch_Pipe`  
**IFC class:** `IfcPipeSegment`

Creates a circular pipe along a Draft Wire or edge. Used for MEP plumbing,
HVAC ductwork (round), and drainage modelling. Pipe fittings at junctions
are added with [Pipe Connector](pipe-connector.md).

---

## Key properties

| Property | Description |
|----------|-------------|
| `Diameter` | Pipe outer diameter (mm) |
| `WallThickness` | Pipe wall thickness (mm) |
| `Base` | Reference wire for the pipe centreline |
