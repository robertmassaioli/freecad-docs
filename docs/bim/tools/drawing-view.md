# Drawing View

> **In one sentence:** Add a 2-D view of a Section Plane's cut geometry to
> the active TechDraw drawing page.

---

## Overview

**Workbench:** BIM  
**Menu:** Annotation → Drawing View  
**Command:** `BIM_TDView`

Adds a view of a [Section Plane](section-plane.md)'s cut geometry to the
active TechDraw page. The view includes cut lines (the cross-section at the
cut plane), visible edges behind the cut plane, and optionally hidden lines.

---

## Step-by-step

1. Create a [Section Plane](section-plane.md) and set its `Objects`.
2. Create a [Drawing Page](drawing-page.md).
3. Select the Section Plane.
4. Run **Annotation → Drawing View**.
5. The 2-D view appears on the TechDraw page.
