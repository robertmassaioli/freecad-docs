# Leader

> **In one sentence:** Create a leader line — an arrow pointing to a model
> feature with a text label at the far end.

---

## Overview

**Workbench:** BIM  
**Menu:** Annotation → Leader  
**Command:** `BIM_Leader`

Creates a leader line — an arrowed line that points to a model feature, with
a text label at the tail end. Used for notes, part callouts, and material
specifications.

---

## When to use it

- Callout annotations pointing to specific elements.
- Material or finish specifications attached to a surface.
- Room name labels with a pointer.
