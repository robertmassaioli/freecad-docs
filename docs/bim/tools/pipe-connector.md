# Pipe Connector

> **In one sentence:** Create a pipe fitting (elbow, tee, or transition) at
> the junction of two or three pipe segments.

---

## Overview

**Workbench:** BIM  
**Menu:** 3D/BIM → Pipe Connector  
**Command:** `Arch_PipeConnector`  
**IFC class:** `IfcPipeFitting`

Creates a fitting (elbow, tee, or reducer) at the junction of two or three
[Pipe](pipe.md) segments. Select the pipes to join, then run the command —
FreeCAD generates the appropriate connector geometry at the intersection.

---

## When to use it

- Elbows at pipe bends.
- Tee junctions where a branch joins the main run.
- Transitions between pipe diameters.
