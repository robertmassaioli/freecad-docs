# Drawing Page

> **In one sentence:** Create a TechDraw page using a standard architectural
> sheet template, ready for BIM views and annotations.

---

## Overview

**Workbench:** BIM  
**Menu:** Annotation → Drawing Page  
**Command:** `BIM_TDPage`

Creates a new TechDraw page using a standard architectural sheet template
(A0–A4 with title block). This is equivalent to TechDraw's Page command but
pre-selects a BIM-appropriate template with an architectural title block.

---

## When to use it

- Creating construction documents (floor plans, sections, elevations).
- Producing a sheet layout to place [Drawing Views](drawing-view.md) on.
