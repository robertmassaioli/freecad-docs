# Frame

> **In one sentence:** Create a structural frame member by sweeping a profile
> along a wire skeleton path.

---

## Overview

**Workbench:** BIM  
**Menu:** 3D/BIM → Frame  
**Command:** `Arch_Frame`  
**IFC class:** `IfcMember`

Creates a structural frame member by sweeping a cross-section profile along
a wire skeleton. The skeleton can be any wire object (Draft Wire, edge, etc.).
Use this for glazing frames, structural members, purlins, and similar linear
elements.

---

## When to use it

- Structural steel or timber frame members that follow arbitrary paths.
- Glazing sub-frames and curtain wall mullions (custom profiles).
- Any profile-sweep element that is not a simple [Beam](beam.md) or [Column](column.md).
