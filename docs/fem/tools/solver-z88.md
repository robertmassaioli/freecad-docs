# Solver Z88

> **In one sentence:** Add a Z88 solver for fast linear static structural
> analysis as an alternative to CalculiX.

---

## Overview

**Workbench:** FEM  
**Menu:** FEM → Solve → Solver Z88

Adds a Z88 solver for linear static structural analysis. Z88 is a lightweight
open-source structural solver. Use it when a fast linear static analysis is
needed and CalculiX is unavailable.

---

## When to use Z88

- Quick linear static structural checks.
- Alternative when CalculiX is not installed.
