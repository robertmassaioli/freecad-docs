# Solver Mystran

> **In one sentence:** Add a Mystran solver for fast, lightweight linear
> static structural analysis — a NASTRAN-derived open-source solver.

---

## Overview

**Workbench:** FEM  
**Menu:** FEM → Solve → Solver Mystran

Adds a Mystran solver for linear static structural analysis. Mystran is a
lightweight, fast open-source structural solver derived from NASTRAN. Use
it for quick linear structural checks when CalculiX is not installed.

---

## Capabilities

- Linear static structural analysis
- Shell and beam elements
- NASTRAN-format bulk data input

---

## When to use Mystran

- Quick linear static checks.
- When CalculiX is not available.
- When a NASTRAN-format workflow is preferred.
