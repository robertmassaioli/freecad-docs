# Pressure Load

> **In one sentence:** Apply a uniform pressure (N/mm² = MPa) perpendicular
> to selected faces — positive pressure pushes inward.

---

## Overview

**Workbench:** FEM  
**Menu:** FEM → Model → Mechanical Loads → Pressure Load

Applies a uniform pressure perpendicular to selected faces. Pressure is always
in the face inward-normal direction. Use a negative value for suction (outward
tension).

---

## Common reference values

| Situation | Pressure |
|-----------|---------|
| 1 bar water | 0.1 MPa |
| 10 bar hydraulic system | 1.0 MPa |
| 100 psi (typical tire) | 0.69 MPa |
| Atmospheric | 0.1013 MPa |
