# FEM Examples

> **In one sentence:** Opens a library of pre-built FEM example analyses
> that demonstrate typical solver setups and can be used as starting templates.

---

## Overview

**Workbench:** FEM  
**Menu:** FEM → Utilities → FEM Examples

Opens the FEM Examples browser. Each example is a complete FEM analysis —
mesh, materials, boundary conditions, and (optionally) pre-computed results —
ready to open and inspect or re-run. Examples cover the main solver types
and physics available in the FEM workbench.

---

## How to use

1. Choose **FEM → Utilities → FEM Examples**.
2. Browse the example list and select a category (structural, thermal,
   electromagnetic, etc.).
3. Double-click an example to open it as a new document.
4. Inspect the model tree, boundary conditions, and material settings.
5. Optionally run the solver to reproduce the results.

---

## Use cases

- **Learning:** Understand how a specific solver or equation is set up by
  examining a working example.
- **Template:** Copy the relevant parts of an example into your own analysis.
- **Verification:** Re-run the example and compare results to the expected
  values to confirm your FreeCAD installation is working correctly.
