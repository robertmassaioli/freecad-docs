# Equation: Magnetodynamic 2D

> **In one sentence:** 2-D (planar or axisymmetric) version of the
> Magnetodynamic equation — faster to solve, sufficient for motor and
> transformer cross-sections.

---

## Overview

**Workbench:** FEM  
**Menu:** FEM → Solve → Elmer Equations → Magnetodynamic 2D

The 2-D (planar or axisymmetric) version of the [Magnetodynamic Equation](equation-magnetodynamic.md).
Faster to solve and sufficient for geometries that are constant in the
out-of-plane direction — motor cross-sections, transformer cores, 2-D coil
problems.
