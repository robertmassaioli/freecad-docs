# Create Elements Set

> **In one sentence:** Define a named set of individual mesh elements by ID
> for selective post-processing or constraint application.

---

## Overview

**Workbench:** FEM  
**Menu:** FEM → Mesh → Create Elements Set

Defines a named set of individual mesh elements by element ID. Unlike
[FEM Mesh Group](mesh-group.md) (which is geometry-based), an Elements Set
is a manual selection of specific elements already in the mesh.

---

## When to use it

- Fine-grained control over which elements appear in a result extraction.
- Selective material zone assignment.
- Identifying a specific subset of elements for output.
