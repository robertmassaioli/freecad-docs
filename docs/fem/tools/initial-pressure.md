# Initial Pressure

> **In one sentence:** Set the starting pressure field for a transient fluid
> flow analysis or provide an initial guess for steady-state convergence.

---

## Overview

**Workbench:** FEM  
**Menu:** FEM → Model → Fluid BCs → Initial Pressure

Sets the starting pressure field for a transient fluid flow analysis using
Elmer. For an incompressible flow with a known inlet pressure, setting an
Initial Pressure that matches the inlet BC speeds up convergence.
