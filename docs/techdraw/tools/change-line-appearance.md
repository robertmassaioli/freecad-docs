# Change Appearance of Lines

> **In one sentence:** Changes the style, width, and colour of selected
> cosmetic lines or view edges via a dialog with line style, weight, and
> colour pickers.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Add Lines → Change Appearance of Lines

Select one or more cosmetic lines or edges, then run this command to change
their visual properties. The dialog provides:

- **Line style** — solid, dashed, center, phantom, dotted, etc.
- **Line width** — weight in mm
- **Colour** — colour picker
