# Add Centerline Between 2 Lines

> **In one sentence:** Draws a centerline equidistant between two selected
> straight edges — for marking symmetry axes and mid-planes.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Add Lines → Add Centerline Between 2 Lines

The centerline is placed halfway between the two selected edges, in
long-dash center-line ISO style. Used to show the symmetry axis of a
symmetric feature or the mid-plane between two parallel faces.

---

## Step-by-step

1. Ctrl-click two straight edges in a view.
2. Choose **TechDraw → Add Lines → Add Centerline Between 2 Lines**.
3. Set extension and flip options.
4. Click **OK**.
