# Insert SVG Symbol

> **In one sentence:** Inserts an SVG file as a static, non-parametric symbol
> on the drawing page — for company logos, north arrows, or general-note
> blocks.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Views → Insert SVG Symbol

An SVG symbol is embedded in the document. You can move and scale it on the
page. Unlike a view, it does not update when the 3-D model changes — it is
pure drawing artwork.

---

## Common uses

- Company logo in the title block area
- North arrow on survey drawings
- General notes block with standard text
- Any vector artwork that does not need to update with the model
