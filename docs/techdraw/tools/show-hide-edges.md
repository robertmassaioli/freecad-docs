# Show / Hide Invisible Edges

> **In one sentence:** Toggles the visibility of individual hidden edges in a
> view — for selectively showing specific hidden edges independent of the
> view-wide setting.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Add Lines → Show / Hide Invisible Edges

By default, hidden edges can be shown as dashed lines if **ShowHidden** is
true on the view. This tool provides per-edge control: you can show a specific
hidden edge while keeping others hidden, or vice versa.

---

## When to use it

- A specific hidden edge (e.g. the back edge of a bore) needs to be called
  out by a dimension, but showing all hidden edges would clutter the view.
- A hidden edge should be suppressed for clarity even when ShowHidden is on.
