# Share View

> **In one sentence:** Makes an existing view appear on a second page without
> duplicating it — both pages show the same live view object.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Views → Share View

A shared view is a single view object that is referenced from multiple pages.
Changes to the view's properties (direction, scale, visibility settings) apply
everywhere it appears. The view does not need to be created and maintained
separately on each page.

---

## When to use it

- A title sheet that shows a key isometric view alongside the detail pages.
- A common view that must appear identically on multiple sheets.
