# Insert Bitmap Image

> **In one sentence:** Inserts a raster image (PNG, JPG, BMP) on the drawing
> page — for photographs, scanned signatures, or content that cannot be
> represented as vector geometry.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Views → Insert Bitmap Image

The image is embedded in the document. You can move and scale it on the page.
Like an SVG symbol, a bitmap image does not update with the 3-D model.

---

## Common uses

- Photographs of a physical prototype
- Scanned signatures or stamps
- QR codes or barcodes
- Screen captures for reference
