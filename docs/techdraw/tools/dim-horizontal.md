# Insert Horizontal Dimension

> **In one sentence:** Measures only the horizontal component of the distance
> between two points — the dimension line is always drawn horizontally.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Dimensions → Insert Horizontal Dimension

Regardless of the actual edge angle, this dimension reports only the
horizontal (X) distance. Use when the drawing standard requires the
horizontal extent to be called out separately (e.g. horizontal position of
a hole centre from a datum edge).

---

## Step-by-step

1. Click two **vertices** in a view.
2. Choose **TechDraw → Dimensions → Insert Horizontal Dimension**.
3. Click to place the dimension line above or below the geometry.
