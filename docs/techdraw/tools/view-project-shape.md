# Project Shape

> **In one sentence:** Projects 3-D edges and wires directly onto the drawing
> page as free-standing 2-D paths — without a view container, and without
> updating when the model changes.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Views → Project Shape

Unlike a normal view, a projected shape is not parametric — it is a one-time
projection that produces static 2-D SVG paths. These paths can be styled and
moved freely.

---

## When to use it

- One-off reference geometry that doesn't need to track model changes.
- Custom diagram elements or decorative lines.
- Geometric construction that needs to live at the drawing level, not the
  model level.
