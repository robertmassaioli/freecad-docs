# Insert Complex Section

> **In one sentence:** Creates an offset or aligned section view that follows
> a bent or stepped cut path — for when a single flat cut would miss important
> features.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Views → Insert Complex Section

A complex section is used when the geometry of interest does not lie on a
single straight plane. Two types are available:

---

## Section types

**Offset Section** — the cut plane is stepped: it starts at one plane, jumps
to a parallel plane, then continues. Each step is perpendicular to the view
direction. Used to catch holes or features at different horizontal positions.

**Aligned Section** — the cut plane is hinged: it consists of multiple planes
that meet at a common axis, like a broken wing. Each segment is rotated into
the projection plane for display. Used for radially-arranged features (bolt
holes on a flange).

---

## Step-by-step

1. Select the parent view on the page.
2. Choose **TechDraw → Views → Insert Complex Section**.
3. In the task panel, select the section type (Offset or Aligned).
4. Draw the cut path by clicking points on the parent view.
5. Click **OK**. The section view appears with the correct projection.
