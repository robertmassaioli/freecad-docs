# Move View to Top of Stack

> **In one sentence:** Renders the selected view on top of all other
> overlapping views on the page.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Stacking → Move View to Top of Stack

When views or annotations overlap, the stacking order determines which one
appears in front. This command brings the selected view to the top of the
z-order. Stacking does not affect the 3-D model.
