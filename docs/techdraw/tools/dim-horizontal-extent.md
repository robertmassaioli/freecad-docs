# Insert Horizontal Extent Dimension

> **In one sentence:** Measures the total horizontal span of all selected
> edges or the entire view — from the leftmost to the rightmost point.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Dimensions → Insert Horizontal Extent Dimension

Creates a dimension from the leftmost to the rightmost point of the selection,
with the dimension line placed below (or above) the view.

---

## When to use it

Use for overall width callouts on complex profiles where no single edge spans
the full width.
