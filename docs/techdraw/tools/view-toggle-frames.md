# Toggle Frames

> **In one sentence:** Shows or hides the blue rectangular frames that surround
> each view on the page — toggle them off to preview the clean drawing output.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Views → Toggle Frames

View frames are a UI aid — they show view boundaries and allow selection by
clicking on the frame edge. They do not appear in exported SVG or DXF files.

Toggle Frames off to see the clean drawing output exactly as it will be
exported or printed.
