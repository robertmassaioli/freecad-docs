# Move View Up One Level

> **In one sentence:** Moves the selected view up one position in the stacking
> z-order.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Stacking → Move View Up One Level

Increments the z-order by one step. Use this for fine-grained control of
overlapping views when the full "top" or "bottom" commands would be too
extreme.
