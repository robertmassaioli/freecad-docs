# Apply Geometric Hatch

> **In one sentence:** Fills a face using a PAT file — the industry-standard
> AutoCAD-compatible hatch format — for patterns defined by ANSI, ISO, or
> company standards.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Hatching → Apply Geometric Hatch

PAT files define hatch patterns as line families with angle, spacing, and
dash parameters. Use Geometric Hatch when you need exact AutoCAD-compatible
patterns or patterns defined by your CAD standard as a `.pat` file.

FreeCAD ships PAT files in `<FreeCAD install>/Mod/TechDraw/PAT/`.

---

## Step-by-step

1. Click a face in a view.
2. Choose **TechDraw → Hatching → Apply Geometric Hatch**.
3. Select the PAT file and the pattern name within it.
4. Set scale and rotation.
5. Click **OK**.
