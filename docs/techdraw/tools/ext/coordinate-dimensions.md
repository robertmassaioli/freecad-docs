# Coordinate Dimensions

> **In one sentence:** Creates baseline ordinate dimensions — all features
> measured from a single datum — so tolerance does not accumulate across
> the drawing.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Format/Organize Dimensions

Three tools:

| Tool | Description |
|------|-------------|
| Create Horizontal Coordinate Dimension | Baseline ordinate — all measured from one datum |
| Create Vertical Coordinate Dimension | Vertical baseline ordinate |
| Create Oblique Coordinate Dimension | Oblique baseline ordinate |

**Coordinate dimensions** (ordinate dimensioning) measure all features from
a single datum baseline. Preferred for CNC machined parts because each
feature is measured independently — there is no accumulation of tolerance
errors as in chain dimensions.

---

## Step-by-step

1. Select the datum vertex first, then all other vertices in order.
2. Choose **Create Horizontal Coordinate Dimension**.
3. The datum is automatically assigned zero; all other dimensions measure
   from it.
