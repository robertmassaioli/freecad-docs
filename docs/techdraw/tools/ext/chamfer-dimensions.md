# Chamfer Dimensions

> **In one sentence:** Dimensions a chamfer line showing the horizontal or
> vertical distance in the standard `C × 45°` or `size × angle` format.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Format/Organize Dimensions

Two tools:

| Tool | Description |
|------|-------------|
| Create Horizontal Chamfer Dimension | Dimension a chamfer with the horizontal distance |
| Create Vertical Chamfer Dimension | Dimension a chamfer with the vertical distance |

Click the two endpoints of a chamfer line. The tool creates a dimension with
the chamfer size in the standard format.
