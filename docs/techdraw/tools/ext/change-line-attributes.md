# Change Line Attributes

> **In one sentence:** Applies the currently saved line attributes (from
> Select Line Attributes) to all selected cosmetic lines — for bulk restyling.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Attributes/Modifications → Change Line Attributes

Select the lines first, then run this command. All selected cosmetic lines
are updated to the current saved style, width, and colour.
