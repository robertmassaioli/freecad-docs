# Add Vertex at Intersection

> **In one sentence:** Adds a cosmetic vertex at the geometric intersection of
> two selected edges — for dimensioning to a filleted corner where the sharp
> intersection no longer exists as a vertex.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Centerlines/Threading → Add Vertex at Intersection

When a corner has been filleted, the original sharp corner vertex no longer
exists. This tool extends the two adjacent edges and adds a cosmetic vertex
at their computed intersection point, which can then be used as a dimension
reference.
