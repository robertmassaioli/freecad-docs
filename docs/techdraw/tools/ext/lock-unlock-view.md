# Lock / Unlock View

> **In one sentence:** Prevents a view from being accidentally moved on the
> page when locked — a padlock icon indicates locked status.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Attributes/Modifications → Lock / Unlock View

Lock views once the layout is finalised to prevent accidental displacement
while adding dimensions and annotations. The padlock icon appears on the
view frame when locked.
