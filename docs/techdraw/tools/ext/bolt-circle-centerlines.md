# Add Bolt Circle Centerlines

> **In one sentence:** Draws the pitch circle diameter (PCD) and radial
> centerlines for a bolt-hole pattern from a selection of all the bolt holes.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Centerlines/Threading → Add Bolt Circle Centerlines

Select all the bolt holes in the pattern at once. The tool computes the
common circle centre, draws the PCD arc, and adds one radial centerline per
hole.
