# Increase / Decrease Decimal Places

> **In one sentence:** Adds or removes one decimal place from all selected
> dimensions in one step — for ensuring consistent decimal precision across
> the drawing.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Format/Organize Dimensions

Two tools:

| Tool | Description |
|------|-------------|
| Increase Decimal Places | Add one decimal place to all selected dimensions |
| Decrease Decimal Places | Remove one decimal place from all selected dimensions |

These tools modify the `FormatSpec` printf precision specifier.

---

## Step-by-step

1. Select multiple dimensions at once (click first, Ctrl-click remaining).
2. Apply **Increase Decimal Places** or **Decrease Decimal Places**.
3. All selected dimensions update simultaneously.
