# Add Circle Centerlines

> **In one sentence:** Draws a crossing centerline (+ cross) through the
> centre of selected circles or arcs in standard long-dash center-line style.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Centerlines/Threading → Add Circle Centerlines

The lines extend beyond the circle by the standard overhang amount. Select
multiple circles at once to add centerlines to all of them in one step.

---

## Step-by-step

1. Click one or more circles in a view (or window-select all at once).
2. Choose **TechDraw → Centerlines/Threading → Add Circle Centerlines**.
3. Centerlines appear on all selected circles.
