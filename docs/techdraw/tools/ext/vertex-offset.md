# Add Offset Vertex

> **In one sentence:** Adds a cosmetic vertex offset from an existing vertex
> by a specified X, Y distance — for dimensioning to a theoretical point.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Centerlines/Threading → Add Offset Vertex

Use for dimensioning to a theoretical point that is a known offset from a
physical corner — for example, a datum point that is 5 mm above and 3 mm
left of a specific vertex.
