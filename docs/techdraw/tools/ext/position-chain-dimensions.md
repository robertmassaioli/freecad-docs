# Position Horizontal / Vertical / Oblique Chain Dimension

> **In one sentence:** Aligns a set of selected dimensions so their dimension
> lines are exactly co-linear — forming a horizontal, vertical, or oblique
> chain.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Attributes/Modifications → Position Horizontal/Vertical/Oblique Chain Dimension

Select all the dimensions in the chain first, then run the command. The
dimensions snap to a common baseline.
