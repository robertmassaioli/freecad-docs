# Cascade Horizontal / Vertical / Oblique Dimensions

> **In one sentence:** Distributes selected dimensions at equal spacing along
> the horizontal, vertical, or oblique axis — producing the standard stacked
> parallel dimension layout.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Attributes/Modifications → Cascade Horizontal/Vertical/Oblique Dimension

Cascading produces the stacked parallel dimension layout commonly seen in
mechanical drawings, where dimensions are evenly spaced away from the part.

---

## Step-by-step

1. Add all the dimensions first.
2. Select them all (click first, Ctrl-click remaining).
3. Run **Cascade Horizontal Dimensions** (or Vertical/Oblique as needed).
4. Set the spacing value in the task panel.
5. Click **OK**. The dimensions redistribute evenly.
