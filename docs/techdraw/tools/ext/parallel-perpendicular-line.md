# Draw Parallel / Perpendicular Line

> **In one sentence:** Draws a cosmetic line parallel or perpendicular to a
> selected edge, passing through a clicked point.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Centerlines/Threading

Two tools:

| Tool | Description |
|------|-------------|
| Draw Parallel Line | Cosmetic line parallel to the selected edge |
| Draw Perpendicular Line | Cosmetic line perpendicular to the selected edge |

Click the reference edge, then click the point through which the new line
should pass. The new cosmetic line is drawn in the current saved line style.
