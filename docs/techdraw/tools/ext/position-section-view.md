# Position Section View

> **In one sentence:** Forces a section view to align exactly horizontally or
> vertically with its parent view — for restoring correct projection alignment
> after a manual move.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Attributes/Modifications → Position Section View

Section views should align with their parent view in the standard projection
relationship. If a section view has been manually moved and drifted out of
alignment, this command snaps it back to the correct position.
