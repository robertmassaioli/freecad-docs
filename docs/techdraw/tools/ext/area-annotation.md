# Calculate Area Annotation / Arc Length Annotation

> **In one sentence:** Reads the area of a selected face or the arc length of
> a selected edge and places the computed value as a text annotation on the view.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Attributes/Modifications → Calculate Area Annotation / Calculate Arc Length Annotation

The annotation updates automatically if the geometry changes. The value is
placed as a text annotation directly on the selected face or edge.
