# Create Arc Length Dimension

> **In one sentence:** Dimensions the arc length of a selected curved edge —
> the distance along the curve, not the chord — with the arc-length symbol (~).

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Format/Organize Dimensions → Create Arc Length Dimension

The arc-length symbol (~) is added automatically. This is distinct from a
radius or chord-length dimension.
