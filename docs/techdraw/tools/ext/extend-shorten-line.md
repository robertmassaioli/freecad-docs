# Extend Line / Shorten Line

> **In one sentence:** Lengthens or shortens a selected cosmetic line by a
> fixed amount at one or both ends.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Attributes/Modifications → Extend Line / Shorten Line

Enter the delta length in the task panel. The extension can be applied to
one end or both ends.

---

## Common use

Extend a centerline so it protrudes beyond the part boundary by the standard
amount required by your drawing standard. ISO requires 2–5 mm overhang for
centerlines.
