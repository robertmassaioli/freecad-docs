# Insert Vertical Dimension

> **In one sentence:** Measures only the vertical component of the distance
> between two points — the dimension line is always drawn vertically.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Dimensions → Insert Vertical Dimension

Regardless of the actual edge angle, this dimension reports only the
vertical (Y) distance. Use for vertical extents, hole heights, or datum
callouts along the vertical axis.

---

## Step-by-step

1. Click two **vertices** in a view.
2. Choose **TechDraw → Dimensions → Insert Vertical Dimension**.
3. Click to place the dimension line to the left or right of the geometry.
