# Add Cosmetic Vertex

> **In one sentence:** Adds a construction point on a view at a position you
> click — as a snap target for dimensions and cosmetic lines.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Add Vertices → Add Cosmetic Vertex

Cosmetic vertices appear as small crosses or dots on the drawing. They exist
only on the drawing — not in the 3-D model — and can be used as snap targets
for dimensions and cosmetic line endpoints.

---

## Common uses

- Marking the centre of a circle for a Radius Dimension where no vertex
  exists at the centre.
- Marking a specific point for a Length Dimension that has no underlying
  vertex.
- Providing a dimensionable snap point for a cosmetic construction line.
