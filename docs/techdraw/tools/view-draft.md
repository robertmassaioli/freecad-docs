# Insert Draft View

> **In one sentence:** Projects Draft workbench objects (wires, shapes, text,
> dimensions) onto a TechDraw page at a specified scale.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Views from Other Workbenches → Insert Draft View

Draft objects are 2-D by nature. The Draft View renders them on a TechDraw
page, allowing 2-D Draft sketches to be included in a formal engineering drawing.

---

## When to use it

- Including a 2-D schematic or outline drawing made in the Draft workbench.
- Adding Draft annotations or dimension objects to a TechDraw page.
- Combining Draft and PartDesign geometry on the same drawing sheet.
