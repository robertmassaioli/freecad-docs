# Insert Spreadsheet View

> **In one sentence:** Embeds a Spreadsheet workbench table on the drawing
> page — the table renders as a cell grid and updates automatically when the
> spreadsheet changes.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Views from Other Workbenches → Insert Spreadsheet View

The spreadsheet view renders exactly as it appears in the Spreadsheet editor.
It is a live link — changes to the spreadsheet automatically update the table
on the drawing.

---

## Common use cases

- Parts list / bill of materials table on the same sheet as a projection group.
- Revision history table in the drawing.
- Parameter table showing design variables.
