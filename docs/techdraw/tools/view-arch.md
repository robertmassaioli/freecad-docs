# Insert Arch View

> **In one sentence:** Projects BIM / Arch workbench objects (walls, floors,
> spaces, sections) onto a TechDraw page — for architectural plan and section
> drawings.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Views from Other Workbenches → Insert Arch View

The Arch View tool is designed for architectural plan and section drawings,
respecting Arch Section Plane objects. It renders BIM geometry in the
conventions expected for architectural drawings.
