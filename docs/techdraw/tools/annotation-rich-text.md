# Insert Rich Text Annotation

> **In one sentence:** Places a text block with basic HTML-like formatting
> (bold, italic, underline, font size) — for notes that need emphasis or
> mixed styles.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Annotations → Insert Rich Text Annotation

The rich text editor opens when you double-click the annotation after placing
it. Supported tags include `<b>`, `<i>`, `<u>`, and basic `<font>` attributes.

---

## Common mistakes and pitfalls

!!! warning "Rich text HTML is limited"
    Only a subset of HTML is supported — basic tags like `<b>`, `<i>`, `<u>`,
    `<font size>`. Full CSS or complex HTML is not supported. For complex
    layouts, use multiple annotations or an SVG symbol.
