# Insert Clip Group

> **In one sentence:** Creates a rectangular clipping frame on the page —
> views placed inside the clip group are cropped to the frame's bounds.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Views → Insert Clip Group

A clip group acts as a rectangular viewport mask. Views placed inside the
clip group are rendered only within the frame's bounds — geometry outside
is hidden. Useful when a view is large but only a portion needs to be visible.
