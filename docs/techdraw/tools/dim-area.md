# Insert Area Annotation

> **In one sentence:** Calculates and displays the area of a selected face in
> the view as a text annotation placed on the face.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Dimensions → Insert Area Annotation

The value appears as a text annotation (not a dimension line) placed on the
selected face. Units follow the dimension unit settings (mm² by default).
