# Remove Cosmetic Objects

> **In one sentence:** Deletes selected cosmetic vertices, cosmetic lines, or
> centerlines from a view.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Remove Cosmetic Objects

Select the cosmetic objects in the view before running this tool.
