# Print All Pages

> **In one sentence:** Sends every TechDraw page in the document to the
> system print dialog in one operation — for multi-sheet drawing packages.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Page → Print All Pages

When a document contains multiple drawing pages (assembly drawing, detail
sheets, revision sheets), this command prints them all in one pass without
needing to activate each page individually.

---

## Printing individual pages

For individual pages, use one of:

- **File → Print** with the drawing page active.
- Right-click the page in the model tree and choose **Print**.
