# Insert Dimension

> **In one sentence:** The smart dimension tool — select geometry and it
> automatically picks the most appropriate dimension type (length, radius,
> diameter, or angle).

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Dimensions → Insert Dimension

Insert Dimension analyses what you have selected and creates the most
appropriate dimension:

- Two vertices → Length Dimension
- One edge (straight) → Length Dimension
- One edge (arc or circle) → Radius or Diameter Dimension
- Two straight edges → Angle Dimension

This is the quickest way to dimension a drawing when you do not need to
specify the type explicitly.

---

## Step-by-step

1. Click a vertex, edge, or pair of geometry in a view.
2. Choose **TechDraw → Dimensions → Insert Dimension**.
3. The dimension appears on the view. Click to place the dimension text
   in a comfortable position.
4. The dimension is now live — it updates when the model changes.
