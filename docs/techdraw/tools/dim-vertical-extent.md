# Insert Vertical Extent Dimension

> **In one sentence:** Measures the total vertical span of all selected edges
> or the entire view — from the bottommost to the topmost point.

---

## Overview

**Workbench:** TechDraw  
**Menu:** TechDraw → Dimensions → Insert Vertical Extent Dimension

Creates a dimension from the bottommost to the topmost point of the selection,
with the dimension line placed to the left (or right) of the view.

---

## When to use it

Use for overall height callouts on complex profiles where no single edge spans
the full height.
