# Map on Face

> **In one sentence:** Maps edges, wires, or shapes onto a target face —
> the more general counterpart to Sketch on Surface that accepts any edge
> object.

---

!!! note "Third-party addon — installation required"
    The Curves workbench is not included in FreeCAD. Install it via
    **Tools → Addon Manager → search "Curves" → Install** and restart FreeCAD.

---

## Overview

**Workbench:** Curves  
**Menu:** Surfaces → Map on Face  
**Command:** `Curves_MapOnFace`

Unlike [Sketch on Surface](sketch-on-surface.md) which works exclusively with
Sketcher sketches, Map on Face accepts any edge or wire object.

---

## Step-by-step

1. Select the objects to map (edges, wires, or shapes).
2. Ctrl-click the target face.
3. Choose **Surfaces → Map on Face**.
