# Blend Solid

> **In one sentence:** Creates a solid blend (fillet-like transition solid)
> between two faces — the solid counterpart to Blend Surface.

---

!!! note "Third-party addon — installation required"
    The Curves workbench is not included in FreeCAD. Install it via
    **Tools → Addon Manager → search "Curves" → Install** and restart FreeCAD.

---

## Overview

**Workbench:** Curves  
**Menu:** Surfaces → Blend Solid  
**Command:** `Curves_BlendSolid`

Similar to [Blend Surface](blend-surface.md) but the result is a solid body
rather than a surface, making it suitable for adding material between two
solid faces.
