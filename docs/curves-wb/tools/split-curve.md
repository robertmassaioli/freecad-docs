# Split Curve

> **In one sentence:** Splits an edge at one or more parameter positions,
> producing multiple sub-edges.

---

!!! note "Third-party addon — installation required"
    The Curves workbench is not included in FreeCAD. Install it via
    **Tools → Addon Manager → search "Curves" → Install** and restart FreeCAD.

---

## Overview

**Workbench:** Curves  
**Menu:** Curves → Split Curve  
**Command:** `split`

Split points can be:
- At a specified parameter value (0–1 along the curve)
- At the nearest point to a selected vertex

---

## Step-by-step

1. Select an edge in the 3-D view.
2. Choose **Curves → Split Curve**.
3. In the task panel, add split points interactively.
4. Click **OK**. The edge is replaced with multiple sub-edges.
