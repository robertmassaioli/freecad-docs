# Gordon Profile

> **In one sentence:** Creates an editable profile curve with interactive
> control-point handles — designed as input for the Gordon Surface tool.

---

!!! note "Third-party addon — installation required"
    The Curves workbench is not included in FreeCAD. Install it via
    **Tools → Addon Manager → search "Curves" → Install** and restart FreeCAD.

---

## Overview

**Workbench:** Curves  
**Menu:** Curves → Gordon Profile  
**Command:** `gordon_profile`

The Gordon Profile produces curves that are guaranteed to intersect correctly
with other Gordon Profile curves. Use them as the U-direction and V-direction
families for the [Gordon Surface](gordon-surface.md) tool.
