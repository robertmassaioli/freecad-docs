# Points Tools

Complete reference for all Points workbench tools.

---

| Tool | Description |
|------|-------------|
| [Import Points](import-points.md) | Import a point cloud from an `.asc`, `.pcd`, `.ply`, or `.e57` file |
| [Export Points](export-points.md) | Export a point cloud to `.asc`, `.pcd`, or `.ply` |
| [Convert to Points](convert-to-points.md) | Sample any geometry (mesh or solid) into a point cloud |
| [Structured Point Cloud](structured-point-cloud.md) | Reorder an unstructured cloud into a Width × Height grid |
| [Cut Point Cloud](cut-point-cloud.md) | Draw a lasso polygon to remove points |
| [Merge Point Clouds](merge-point-clouds.md) | Combine multiple point clouds into one |
