# Points Tools

| Tool | Description |
|------|-------------|
| [Import Points](tools.md#import-points) | Import a point cloud from an .asc, .pcd, .ply, or .e57 file |
| [Export Points](tools.md#export-points) | Export a point cloud to .asc, .pcd, or .ply |
| [Convert to Points](tools.md#convert-to-points) | Sample any geometry (mesh or solid) into a point cloud |
| [Structured Point Cloud](tools.md#structured-point-cloud) | Reorder an unstructured cloud into a Width × Height grid |
| [Cut Point Cloud](tools.md#cut-point-cloud) | Draw a lasso polygon to remove points |
| [Merge Point Clouds](tools.md#merge-point-clouds) | Combine multiple point clouds into one |
