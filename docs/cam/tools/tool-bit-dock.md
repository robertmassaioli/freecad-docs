# Tool Bit Dock

> **In one sentence:** Show or hide the compact floating panel for quickly
> adding tools to a job without opening the full library manager.

---

## Overview

**Workbench:** CAM  
**Menu:** CAM → Tool Bit Dock  
**Command:** `CAM_ToolBitDock`  
**Shortcut:** none

Toggles the Tool Bit Dock — a compact floating panel that shows your tool
libraries and lets you drag tools onto a Job or click Add to Job without
going through the full Library manager.

---

## Intuition

The [Tool Bit Library](tool-bit-library.md) manager is the full editor for
managing your tool collection. The Dock is the quick-access panel for the
most common task: picking a tool from a library and adding it to a job while
you are setting up operations.

---

## When to use it

- Quickly adding tools to a job during operation setup.
- Browsing available tools while keeping the main 3D view visible.
- Faster workflow than opening the full Library manager.
