# Dress-up: Z Correct

> **In one sentence:** Apply a probed height map to every Z move in a path,
> compensating for workpiece warp or table non-flatness.

---

## Overview

**Workbench:** CAM  
**Menu:** CAM → Dress-up → Z Correct  
**Command:** `CAM_DressupZCorrect`  
**Shortcut:** none

Reads a Z-height correction map generated by a probing pass and adds the
measured offsets to every Z-axis move in the toolpath. This compensates for
workpiece warp, stock thickness variation, or table non-flatness — keeping
the tool at the correct depth relative to the actual surface.

---

## Intuition

A large PCB or thin sheet workpiece may not be perfectly flat. Even a 0.1 mm
warp across 300 mm can cause etched traces to be too shallow on one end and
too deep on the other. Z Correct uses a grid of probe measurements to build
a height map, then warps the entire toolpath to follow the actual surface.

---

## Typical workflow

1. Run a [Probe](probe.md) operation to measure the workpiece surface at a grid of points.
2. The probe measurements are saved to a correction file.
3. Apply Z Correct dress-up to the milling operation, pointing it at the correction file.
4. The Z coordinates are adjusted automatically.

---

## When to use it

- PCB engraving/milling on slightly warped boards.
- Thin sheet material that is not perfectly flat.
- Any high-precision surfacing operation where workpiece flatness is uncertain.
