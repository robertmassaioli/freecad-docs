    Part Design Tools¶

    Complete reference index for all Part Design workbench tools, grouped by function. Each tool name links to its dedicated page.

    Structure & setup¶

    Tools for creating the model container, attaching sketches, and defining reference geometry that other features snap to.

    Tool Description
    Body Create a new Body — the mandatory container for all Part Design features
    New Sketch Create a sketch attached to a face or datum plane within the active Body
    Datum Plane Insert a parametric reference plane at any position or angle
    Datum Line Insert a parametric reference line (axis)
    Datum Point Insert a parametric reference point
    Coordinate System Insert a local coordinate system (all three datum axes at once)
    Shape Binder Copy external geometry into the Body as a non-solid reference
    Sub-Shape Binder Reference a specific face, edge, or vertex from another Body
    Clone Insert a linked clone of another Body

    Additive features¶

    Sketch-based tools that add material to the Body.

    Tool Description
    Pad Extrude a closed sketch perpendicular to its plane to create a solid
    Revolution Revolve a sketch profile around an axis to create a solid of revolution
    Additive Loft Blend through two or more sketch profiles to create a smooth solid
    Additive Pipe Sweep a sketch profile along a path curve
    Additive Helix Sweep a sketch profile along a helical path

    Additive primitives¶

    Insert parametric solid shapes without a sketch, adding material to the Body.

    Tool Description
    Additive Box Add a rectangular box
    Additive Cylinder Add a cylinder
    Additive Sphere Add a sphere
    Additive Cone Add a cone or truncated cone
    Additive Ellipsoid Add an ellipsoid
    Additive Torus Add a torus (donut shape)
    Additive Prism Add a regular polygonal prism
    Additive Wedge Add a wedge (parametric tapered box)

    Subtractive features¶

    Sketch-based tools that remove material from the Body.

    Tool Description
    Pocket Extrude a cut into the solid — the inverse of Pad
    Groove Revolve a cut around an axis — the inverse of Revolution
    Hole Create an engineered hole with optional thread, counterbore, or countersink
    Subtractive Loft Blend-cut through two or more sketch profiles
    Subtractive Pipe Sweep a cut along a path curve
    Subtractive Helix Sweep a cut along a helical path

    Subtractive primitives¶

    Insert parametric solid shapes that remove material from the Body.

    Tool Description
    Subtractive Box Remove a rectangular box
    Subtractive Cylinder Remove a cylinder
    Subtractive Sphere Remove a sphere
    Subtractive Cone Remove a cone or truncated cone
    Subtractive Ellipsoid Remove an ellipsoid
    Subtractive Torus Remove a torus
    Subtractive Prism Remove a regular polygonal prism
    Subtractive Wedge Remove a wedge

    Dress-up features¶

    Apply finishing operations to edges and faces of an existing solid.

    Tool Description
    Fillet Round selected edges with a constant or variable radius
    Chamfer Bevel selected edges at a specified angle and distance
    Draft Taper selected faces by a draft angle — useful for mould tooling
    Thickness Hollow a solid to a uniform wall thickness by removing selected faces

    Transformation features¶

    Repeat or mirror existing features without redrawing them.

    Tool Description
    Mirrored Mirror one or more features across a plane
    Linear Pattern Repeat features in a straight line with a fixed spacing
    Polar Pattern Repeat features in a circular array around an axis
    Multi-Transform Combine multiple transformation types into a single compound pattern

    Boolean operations¶

    Combine multiple Bodies using Boolean set operations.

    Tool Description
    Boolean Union, subtract, or intersect two Bodies

    Specialised tools¶

    Tool Description
    Involute Gear Generate a parametric involute gear profile sketch
    Sprocket Generate a parametric sprocket profile sketch
    Shaft Wizard Interactive wizard for designing stepped shafts

    Tree management¶

    UI operations for reorganising the feature tree. These are workflow tools, not modelling features; they do not change the final shape, only the tree structure.

    Tool Description
    Move Tip Set the active Tip to any feature in the tree
    Move Feature Move a feature to a different Body
    Move Feature in Tree Reorder a feature within the same Body's tree
    Duplicate Selection Duplicate the selected feature(s) within the Body
